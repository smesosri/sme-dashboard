Panduan Penggunaan Dashboard
1

Upload Data LW321

Dashboard membutuhkan 4 file data LW321. Semua diproses di browser — tidak dikirim ke server.

  • LW Today — Data LW321 hari ini → upload setiap pagi
  • LW MTM — Data tanggal sama bulan lalu → upload setiap hari
  • LW MTD — Akhir bulan lalu → update tiap awal bulan
  • LW YTD — Akhir tahun lalu → update tiap awal tahun
💾 Data ter-cache di browser. Besok tinggal upload LW Today + MTM saja, MTD & YTD masih tersimpan.
2

Format Kolom LW321

Dashboard membaca kolom berikut dari file LW321:

  • UKER — Nama unit kerja
  • NAMA_DEBITUR / NOMOR_REKENING
  • BALANCE DALAM IDR atau PLAFON DALAM IDR — OS debitur
  • KOL_ADK — Kolektibilitas (1/2/3/4/5)
  • PN PENGELOLA SINGLEPN — RM pengelola
  • TGL JATUH TEMPO — untuk menu Jatuh Tempo
  • LN_TYPE — DL/WZ/EB/EO dll
3

Deploy ke GitHub Pages (Akses Online)

Agar bisa diakses semua orang via link, deploy ke GitHub Pages (gratis):

Langkah Deploy

  1. Daftar / login di github.com
  2. New repository → nama sme-dashboard → Create
  3. Upload file ini sebagai index.html
  4. Settings → Pages → Source: branch main → Save
  5. URL aktif: https://brianhayom.github.io/sme-dashboard/
  6. Update konten: ganti file index.html di repository
🔒 Data LW321 hanya ada di browser Anda, tidak disimpan di server GitHub. Aman.
