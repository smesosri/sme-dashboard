Panduan Penggunaan Dashboard
1

Upload Data LW321

Dashboard menggunakan 4 file data LW321 yang diupload secara manual. File diproses di browser — data tidak dikirim ke server.

  • LW Today — Data LW321 hari ini (upload setiap hari pagi)
  • LW MTM — Data LW321 tanggal yang sama bulan lalu (upload setiap hari)
  • LW MTD — Data LW321 akhir bulan lalu (update tiap awal bulan)
  • LW YTD — Data LW321 akhir tahun lalu (update tiap awal tahun)
💡 Format file: .xlsx atau .xls. Struktur kolom harus sama dengan LW321 standar BRI.
2

Dashboard Uker

Menampilkan KPI utama SME dengan filter per unit kerja.

  • Pilih Konsolidasi untuk melihat data gabungan KC + KCP
  • KPI OS, SML, NPL, LAR, LR ditampilkan beserta Gap MTD, MTM, YTD
  • Pencapaian RKA dihitung otomatis jika RKA sudah diinput
  • Warna merah = di bawah target, hijau = di atas target
  • Export ke Excel atau Print langsung dari tombol atas
3

Movement & Kolektibilitas

Analisis pergerakan debitur antar kolektibilitas (Lancar, SML, NPL).

  • Movement — Ringkasan OS movement (realisasi, suplesi, deplesi, dll)
  • KOL Movement — Debitur yang naik/turun kolektibilitas
  • Filter per uker untuk drill-down ke KC atau KCP
  • Klik kategori di tab Kolektibilitas untuk detail debitur per gerakan
4

Input RKA

Form input target RKA yang lebih mudah dari Excel.

  • Input langsung di browser, tersimpan otomatis
  • Import dari file Excel RKA existing
  • Export kembali ke Excel setelah selesai input
  • Nilai dalam Juta Rupiah (Rp 1.000.000)
  • Data RKA dipakai otomatis di Dashboard untuk menghitung pencapaian
5

Cara Akses Online (Deploy ke GitHub Pages) — GRATIS

Agar bisa diakses semua orang tanpa install apapun, deploy file ini ke GitHub Pages:

📋 Langkah Deployment (sekali saja, gratis selamanya)

  1. Buat akun di github.com (gratis)
  2. Klik New repository → beri nama misal sme-dashboard
  3. Centang "Add a README file" → klik Create repository
  4. Upload file index.html ini ke repository (drag & drop)
  5. Masuk ke Settings → Pages
  6. Pada Source, pilih Deploy from a branch → branch main → folder / (root)
  7. Klik Save → tunggu 1-2 menit
  8. URL Anda: https://[username].github.io/sme-dashboard/

Untuk update konten dashboard, cukup ganti file index.html di repository. Data LW321 tetap diupload manual setiap hari — tidak tersimpan di server, privasi terjaga.

🔒 Keamanan Data: Semua data LW321 hanya diproses di browser Anda. Tidak ada data yang dikirim ke internet. Jika browser/tab ditutup, upload ulang file untuk melihat data.

📱 Akses Mobile: Dashboard responsif dan bisa dibuka di HP. Namun untuk upload file, gunakan laptop/PC agar lebih mudah.
