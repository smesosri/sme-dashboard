📖 Panduan Penggunaan Dashboard

📤 Upload Data Harian

Setiap hari, unduh laporan LW321 dari sistem BRI kemudian upload lewat tombol di atas:

1. LW Today → file LW321 hari ini (harian)
2. LW MTM → file LW321 tanggal yang sama bulan lalu (harian)
3. LW MTD → file LW321 akhir bulan lalu (update tiap awal bulan)
4. LW YTD → file LW321 akhir tahun lalu (update tiap awal tahun)

📊 Tab Dashboard

Pilih Unit Kerja di dropdown untuk melihat KPI per Uker, atau pilih Konsolidasi untuk total KC+KCP. Tabel detail menampilkan posisi vs YTD/MTD/MTM/Today beserta gap dan % pencapaian RKA.

↔️ Tab Movement

Melihat pergerakan OS dan kolektibilitas antar periode (MTM comparison).

📋 Tab Kolektibilitas

Detail debitur berdasarkan kategori kolektibilitas. Filter per Uker dan per kategori KOL.

👥 Tab Data Debitur

Seluruh data debitur dengan fitur pencarian nama/rekening dan filter Uker & KOL.

🎯 Tab Input RKA

Input target RKA bulanan untuk setiap KPI. Data tersimpan di browser (localStorage).
